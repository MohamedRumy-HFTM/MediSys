# MediSys - Praxisverwaltungssystem

## Beschreibung
MediSys ist eine JavaFX-basierte Desktop-Anwendung zur Verwaltung von Arztpraxen.
Das System ermöglicht die vollständige Verwaltung von Patienten, Terminen, Behandlungen,
Rechnungen und Medikamenten.

## Technologie-Stack
- **Programmiersprache:** Java 17
- **GUI-Framework:** JavaFX 21
- **Datenbank:** MySQL 8.0
- **Build-Tool:** Maven
- **Passwort-Hashing:** BCrypt

## Voraussetzungen
- Java JDK 17 oder höher
- Maven 3.8 oder höher
- MySQL Server 8.0
- Git
